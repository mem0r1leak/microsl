# microsl (**WIP**) (`msl`) - бібліотека створена для месенджера Citadel

**microsl** — це легка, передбачувана та header-only заміна стандартної бібліотеки шаблонів C++ (STL).

Вона створена для розробників, яким подобається простота, лінійність та повний контроль над пам'яттю мови **C**, але які втомилися від відсутності генериків і постійного дублювання коду.

Бібліотека повністю відмовляється від важких високорівневих абстракцій STL, прихованих алокацій, ітераторного пекла та винятків (`-fno-exceptions`). Тільки плоскі дані, шаблони як копіпаст-машина та максимальний перформанс.

---

## 🎯 Філософія проекту

* 🚫 **Zero Exceptions:** Ніяких `try/catch` та таблиць розкрутки стеку. Помилки — це просто дані (`Result`).
* 📦 **Explicit Memory Management:** Ніяких прихованих `malloc` всередині контейнерів. Кожна алокація явна, через передачу `Allocator*` (Zig/Odin стиль).
* ⚙️ **C with Templates:** Об'єкти — це пласкі структури (POD). Шаблони використовуються виключно для генерації типів на етапі компіляції.
* ⚡ **Bare-metal Ready:** Ідеально підходить для системного програмування, геймдеву та мікроконтролерів, де кожен байт на рахунку.

## 🛠 Що всередині?

Наразі бібліотека надає базові низькорівневі примітиви, які повністю замінюють аналоги з STL:

* **`msl::types` & `concepts`** — заміна `<type_traits>` та базових типів. Фіксовані типи (`i32`, `u64`, `usize`) та залізобетонні концепти (`integral`, `floating_point`, `trivially_copyable`) з очищенням від `const/volatile`.
* **`Option<T>`** — заміна `std::optional`. Безпечний обгортковий тип для значень, які можуть бути відсутні. Реалізований через `union`, не виділяє пам'ять у купі.
* **`Result<OkType, ErrType>`** — заміна виняткам та `std::expected`. Повертає або результат, або помилку. Працює на семантиці переміщення (Move).
* **`Allocator`** — інтерфейс для кастомного керування пам'яттю. Жоден контейнер не володіє пам'яттю сам по собі — виділення та ініціалізація (`placement new`) явно розділені.
* **`Func<Signature>`** — заміна `std::function`. Легкий вказівник на функцію з нормальним синтаксисом і нульовим оверхедом (без динамічної пам'яті).
* **`Span<T>`** — заміна `std::span`. Тупий і швидкий вказівник на масив разом із його розміром.

Бібліотека активно розвивається і може змінювати API та **ламати зворотну сумісність**
